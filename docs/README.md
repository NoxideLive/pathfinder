# Documentation Index

This directory contains documentation for Pathfinder features and development guides.

## Features

### [Multi-Character Tracking](multi-character-tracking.md)
Track multiple characters from a single browser tab with cross-tab coordination and visual indicators.

**Key Features:**
- Character selection interface with tracking checkboxes
- Smart cross-tab prevention to avoid tracking conflicts
- Visual feedback with tracking counter and status indicators
- State persistence across page reloads

## Contributing Documentation

When adding new features or significant changes to Pathfinder, please:

1. Create comprehensive documentation in this `docs/` directory
2. Include both user-facing and technical implementation details
3. Add screenshots or examples where applicable
4. Update this index file with a link to your new documentation
5. Update the main README.md to reference the new documentation

## Documentation Structure

For feature documentation, please include:

- **User Guide**: How to use the feature from an end-user perspective
- **Technical Implementation**: Architecture, file structure, and data flow
- **Developer Guide**: API reference, extension points, and maintenance information
- **Browser Compatibility**: Requirements and known limitations
- **Troubleshooting**: Common issues and solutions
