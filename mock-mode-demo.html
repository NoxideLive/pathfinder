About Mock Mode:
This demo showcases the Pathfinder mock/dev mode feature. Mock mode intercepts AJAX requests and returns mock data, allowing UI development without a backend server.

Key Features:
  • Environment-controlled activation (MOCK_ALLOWED setting)
  • Color-coded console logging
  • Simulated network delays
  • Production-safe (disabled by default in PRODUCTION)

How to Enable Mock Mode:
  1. Ensure MOCK_ALLOWED = 1 in app/environment.ini
  2. Add ?mockMode=true to your URL
  3. Open browser DevTools (F12) and check the Console tab
  4. Look for the red [MOCK MODE ENABLED] banner
